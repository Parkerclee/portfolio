# Learning Locker (local Docker)

This folder runs a local Learning Locker stack for receiving xAPI statements from your portfolio demos.

## Start

From the repo root:

```bash
cp docker/learning-locker/.env.learninglocker.example docker/learning-locker/.env.learninglocker
cp docker/learning-locker/.env.xapi-service.example docker/learning-locker/.env.xapi-service
docker compose -f docker/learning-locker/docker-compose.yml up -d
```

## Create an admin user

```bash
docker exec learninglocker bash -lc 'node ./cli/dist/server createSiteAdmin "admin@example.com" "Portfolio" "password123"'
```

Then open:
- UI: `http://localhost:3000`

## xAPI endpoint (for demos)

Learning Locker’s xAPI HTTP interface is available at:
- `http://localhost:8080/data/xAPI/statements`

Requests must include:
- `Authorization: Basic <base64(clientKey:clientSecret)>`
- `X-Experience-API-Version: 1.0.3`

In the UI, create a Store (LRS) and use the autogenerated Client (or create one) with **xAPI scope: All** and an LRS selected.
